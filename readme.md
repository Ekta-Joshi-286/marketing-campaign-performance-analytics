# Marketing Campaign Performance Analytics Dashboard

## Project Overview

This project focuses on analyzing marketing campaign performance using Python and Streamlit. The dashboard provides insights into campaign effectiveness, customer behavior, marketing ROI, conversion performance, and revenue generation across different channels, regions, and devices.

The objective is to transform raw marketing data into actionable business insights through data cleaning, exploratory data analysis (EDA), KPI creation, and interactive dashboard development.

---

## Business Problem

Marketing teams invest significant budgets across multiple advertising channels, but identifying which campaigns generate the highest return on investment can be challenging.

Key business questions addressed in this project include:

* Which marketing channels generate the highest revenue and ROI?
* Which campaigns drive the most conversions?
* How effectively do users move through the marketing funnel?
* Which customer segments contribute the most revenue?
* How does performance vary across regions and devices?
* Where should marketing budgets be optimized for better results?

This dashboard helps stakeholders make data-driven decisions by providing a centralized view of campaign performance and profitability.

---

## Dataset

* Marketing Analytics Portfolio Dataset
* 50,000+ marketing campaign records
* Includes campaign, customer, channel, region, device, spend, revenue, impressions, clicks, leads, and conversions data

## Project Workflow

### 1. Data Cleaning

* Handled missing values
* Removed duplicate records
* Validated data types
* Prepared data for analysis

### 2. Exploratory Data Analysis (EDA)

* Revenue analysis
* Spend analysis
* Campaign performance analysis
* Channel performance comparison
* Regional and device insights
* Correlation analysis

### 3. Data Preparation

Created business metrics including:

* CTR (Click-Through Rate)
* Conversion Rate
* Lead Rate
* ROI (Return on Investment)
* Profit

### 4. Dashboard Development

Built an interactive multi-page dashboard using Streamlit and Plotly.
